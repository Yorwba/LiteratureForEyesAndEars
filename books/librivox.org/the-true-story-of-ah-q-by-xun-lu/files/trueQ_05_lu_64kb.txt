第五章
生計問題

阿[[Ｑ|| Q ]]禮畢之後﹐仍舊回到土谷祠﹐太陽下去了﹐漸漸覺得世上有些古怪。他仔細一想﹐終於省悟過來﹕其原因蓋在自己的赤膊。他記得破夾襖還在﹐便披在身上﹐躺倒了﹐待張開眼睛﹐原來太陽又已經照在西牆上頭了。他坐起身﹐一面說道：“媽媽的……”

他起來之後﹐也仍舊在街上逛﹐雖然不比赤膊之有切膚之痛﹐卻又漸漸的覺得世上有些古怪了。

彷彿從這一天起﹐未莊的女人們忽然都怕了羞﹐伊們一見阿[[Ｑ|| Q ]]走來﹐便個個躲進門裡去。甚而至於將近五十歲的鄒七嫂﹐也跟著別人亂鑽﹐而且將十一歲的女兒都叫進去了。阿[[Ｑ|| Q ]]很以為奇﹐而且想﹕“這些東西忽然都學起小姐模樣來了。這娼婦們……”

但他更覺得世上有些古怪﹐卻是許多日以後的事。

其一﹐酒店不肯賒欠了﹔其二﹐管土谷祠的老頭子說些廢話﹐似乎叫他走﹔其三﹐他雖然記不清多少日﹐但確乎有許多日﹐沒有一個人來叫他做短工。酒店不賒﹐熬著也罷了﹔老頭子催他走﹐嚕囌一通也就算了﹔只是沒有人來叫他做短工﹐卻使阿[[Ｑ|| Q ]]肚子餓﹕這委實是一件非常〈媽媽的〉的事情。

阿[[Ｑ|| Q ]]忍不下去了﹐他只好到老主顧的家裡去探問﹐──但獨不許踏進趙府的門檻﹐──然而情形也異樣﹕一定走出一個男人來﹐現了十分煩厭的相貌﹐像回覆乞丐一般的搖手道﹕

“沒有沒有﹗你出去﹗”

阿[[Ｑ|| Q ]]愈覺得稀奇了。他想﹐這些人家向來少不了要幫忙﹐不至於現在忽然都無事﹐這總該有些蹊蹺在裡面了。他留心打聽﹐才知道他們有事都去叫小 Don 。

這小[[Ｄ|| D ]]﹐是一個窮小子﹐又瘦又乏﹐在阿[[Ｑ|| Q ]]的眼睛裡﹐位置是在王胡之下的﹐誰料這小子竟謀了他的飯碗去。所以阿[[Ｑ|| Q ]]這一氣﹐更與平常不同﹐當氣憤憤的走著的時候﹐忽然將手一揚﹐喝道﹕“我手執鋼鞭將你打﹗……”

幾天之後﹐他竟在錢府的照壁前遇見了小[[Ｄ|| D ]]。〈仇人相見份外眼明〉﹐阿[[Ｑ|| Q ]]便迎上去﹐小[[Ｄ|| D ]]也站住了。

“畜生﹗”阿[[Ｑ|| Q ]]怒目而視的說﹐嘴角上飛出唾沫來。“我是蟲豸﹐好麼﹖……”小[[Ｄ|| D ]]說。

這謙遜反使阿[[Ｑ|| Q ]]更加憤怒起來﹐但他手裡沒有鋼鞭﹐於是祇得撲上去﹐伸手去拔小[[Ｄ|| D ]]的辮子。

小[[Ｄ|| D ]]一手護住了自己的辮根﹐一手也來拔阿[[Ｑ|| Q ]]的辮子﹐阿[[Ｑ|| Q ]]也……阿[[Ｑ|| Q ]]便將……阿[[Ｑ|| Q ]]便也將空著的一隻手護住了自己的辮根。

從先前的阿[[Ｑ|| Q ]]看來﹐小[[Ｄ|| D ]]本來是不足齒數的﹐但他近來挨了餓﹐又瘦又乏已經不下於小[[Ｄ|| D ]]﹐所以便成了勢均力敵的現象﹐四隻手拔著兩顆頭﹐都彎了腰﹐在錢家粉牆上映出一個藍色的虹形﹐至於半點鐘之久了。

“好了﹐好了﹗”看的人們說﹐大約是解勸的。

“好﹐好﹗”看的人們說﹐不知道是解勸﹐是頌揚﹐還是煽動。

然而他們都不聽。阿[[Ｑ|| Q ]]進三步﹐小[[Ｄ|| D ]]便退三步﹐都站著﹔小[[Ｄ|| D ]]進三步﹐阿[[Ｑ|| Q ]]便退三步﹐又都站。大約半點鐘﹐──未莊少有自鳴鐘﹐所以很難說﹐或者二十分﹐──他們的頭髮裡便都冒煙﹐額上便都流汗﹐阿[[Ｑ|| Q ]]的手放鬆了﹐在同一瞬間﹐小[[Ｄ|| D ]]的手也正放鬆了﹐同時直起﹐同時退開﹐都擠出人叢去。

“記著罷﹐媽媽的……”阿[[Ｑ|| Q ]]回過頭去說。

“媽媽的﹐記著罷……”﹐小[[Ｄ|| D ]]也回過頭來說。

這一場〈龍虎鬥〉似乎並無勝敗﹐也不知道看的人可滿足﹐都沒有發什麼議論﹐而阿[[Ｑ|| Q ]]卻仍然沒有人來叫他做短工。

有一日很溫和﹐微風拂拂的頗有些夏意了﹐阿[[Ｑ|| Q ]]卻覺得寒冷起來﹐但這還可擔當﹐第一倒是肚子餓。棉被﹐氈帽﹐布衫﹐早已沒有了﹐其次就賣了棉襖﹔現在有褲子﹐卻萬不可脫的﹔有破夾襖﹐又除了送人做鞋底之外﹐決定賣不出錢。

他早想在路上拾得一注錢﹐但至今還沒有見﹔他想在自己的破屋裡忽然尋到一注錢﹐慌張的四顧﹐但屋內是空虛而且瞭然。於是他決計出門求食去了。

他在路上走著要〈求食〉﹐看見熟識的酒店﹐看見熟識的饅頭﹐但他都走過了﹐不但沒有暫停﹐而且並不想要。他所求的不是這類東西了﹔他求的是什麼東西﹐他自己不知道。

未莊本不是大村鎮﹐不多時便走盡了。村外多是水田﹐滿眼是新秧的嫩綠﹐夾著幾個圓形的活動的黑點﹐便是耕田的農夫。阿[[Ｑ|| Q ]]並不賞鑒這田家樂﹐卻只是走﹐因為他直覺的知道這與他的〈求食〉之道是很遼遠的。但他終於走到靜修庵的牆外了。

庵週圍也是水田﹐粉牆突出在新綠裡﹐後面的低土牆裡是菜園。阿[[Ｑ|| Q ]]遲疑了一會﹐四面一看﹐並沒有人。他便爬上這矮牆去﹐扯著何首烏藤﹐但泥土仍然簌簌的掉﹐阿[[Ｑ|| Q ]]的腳也索索的抖﹔終於攀著桑樹枝﹐跳到裡面了。

裡面真是鬱鬱蔥蔥﹐但似乎並沒有黃酒饅頭﹐以及此外可吃的之類。靠西牆是竹叢﹐下面許多筍﹐只可惜都是並未煮熟的﹐還有油菜早經結子﹐芥菜已將開花﹐小白菜也很老了。

阿[[Ｑ|| Q ]]彷彿文童落第似的覺得很冤屈﹐他慢慢走近園門去﹐忽而非常驚喜了﹐這分明是一畦老蘿蔔。他於是蹲下便拔﹐而門口突然伸出一個很圓的頭來﹐又即縮回去了﹐這分明是小尼姑。小尼姑之流是阿[[Ｑ|| Q ]]本來視若草芥的﹐但世事須〈退一步想〉﹐所以他便趕緊拔起四個蘿蔔﹐擰下青葉﹐兜在大襟裡。

然而老尼姑已經出來了。

“阿彌陀佛﹐阿[[Ｑ|| Q ]]﹐你怎麼跳進園裡來偷蘿蔔﹗……阿呀﹐罪過呵﹐阿唷﹐阿彌陀佛﹗……”

“我什麼時候跳進你的園裡來偷蘿蔔﹖”阿[[Ｑ|| Q ]]且看且走的說。

“現在……這不是﹖”老尼姑指著他的衣兜。

“這是你的﹖你能叫得他答應你麼﹖你……”

阿[[Ｑ|| Q ]]沒有說完話﹐拔步便跑﹔追來的是一匹很肥大的黑狗。這本來在前門的﹐不知怎的到後園來了。黑狗哼而且追﹐已經要咬著阿[[Ｑ|| Q ]]的腿﹐幸而從衣兜裡落下一個蘿蔔來﹐那狗給一嚇﹐略略一停﹐阿[[Ｑ|| Q ]]已經爬上桑樹﹐跨到土牆﹐連人和蘿蔔都滾出牆外面了。只剩著黑狗還在對著桑樹嗥﹐老尼姑念佛。

阿[[Ｑ|| Q ]]怕尼姑又放出黑狗來﹐拾起蘿蔔便走﹐沿路又檢了幾塊小石頭﹐但黑狗卻並不再出現。阿[[Ｑ|| Q ]]於是拋了石塊﹐一面走一面吃﹐而且想道﹐這裡也沒有什麼東西尋﹐不如進城去……

待三個蘿蔔吃完時﹐他已經打定了進城的主意了。
